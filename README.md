## 学んだこと・疑問に思ったこと 
- fontには様々な形式がある。
- true type：レガシーな形式。数が多く様々なソフトウェアに使われているが容量が小さい。
- open type：true typeに比べて容量が増えた
- woff 上記二つに比べて軽い
- woff2 woffとの違いは圧縮アルゴリズム
- サブセット化してからwoffへ変換することで軽くすることができる。
